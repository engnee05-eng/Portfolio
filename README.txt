EXACT WEBSITE COPY

This folder is the exact compiled website export from the Vite project.
It preserves the same HTML, CSS, JavaScript, assets, and functionality as the production build.

How to open it correctly:

1. Double-click open-website.bat
2. Your browser will open:
   http://127.0.0.1:4173/

Important:

- Do not open index.html directly with file:// if you want the full app behavior.
- For an exact match of the Vite site, it must be served locally.
